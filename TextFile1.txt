Напишите консольное приложение на C#, которое будет 
имитировать систему учёта товаров в магазине. 
В системе должны быть реализованы следующие элементы:

Интерфейс IStoreItem:
Содержит свойства ItemId (уникальный идентификатор товара)
и Price (цена товара).

Класс StoreItem (реализующий IStoreItem):
Содержит конструктор, принимающий ItemId и Price товара.

Класс StoreInventory:
Имеет методы для добавления и удаления товаров, 
а также для поиска товара по ItemId.

Обработка исключений:
Обеспечьте обработку исключений при попытке добавления 
товара с уже существующим ItemId, при поиске несуществующего 
товара и других возможныхисключительных ситуациях.



Генерация данных и LINQ:
Сгенерируйте случайным образом большое количество товаров и 
добавьте их в StoreInventory.
Используйте LINQ для осуществления поиска товаров по заданным 
критериям (например, по цене).

Дополнительные задания:
Реализуйте методы сортировки товаров по различным критериям 
с использованием generics.
Добавьте возможность изменения цены товара.
Создайте отчёт, группирующий товары по ценовым диапазонам.

Примечания:
Обратите внимание на использование интерфейсов для 
обобщения операций с товарами.
Придайте внимание обработке исключений для обеспечения 
безопасности при работе с данными.
Используйте LINQ для более эффективной обработки и 
поиска данных в коллекциях.
